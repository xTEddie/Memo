- Activates tmux shortcut
> [Ctrl+B]

- List tmux sessions
> tmux ls
or
> [Ctrl+B] + s

- Create tmux new session
> tmux new -s session-name
or
> tmux new

- Get in a existing tmux session
> tmux attach

- Change session name
> [Ctrl+B] + $

- Kill tmux session
> tmux kill-session -t [session_name]

- Create window
> [Ctrl+B] + c

- Change window name
> [Ctrl+B] + ,

- List windows
> [Ctrl+B] + w

- Open Previous window
> [Ctrl+B] + p

- Open Next window
> [Ctrl+B] + n

- Split window into column (vertically)
> [Ctrl+B] + %

- Split into row (horizontally)
> [Ctrl+B] + : + "split-window"

- Switch to another pane
> [Ctrl-B] + o

- Kill current pane
> [Ctrl-B] + x

- Activate scroll mode
> [Ctrl-B] + [

- Quit scroll mode
> q
