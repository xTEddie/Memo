- Display system info
> docker info

- Fetch image from Docker registry & Save in system
> docker pull [image]

- List docker images
> docker images

- Remove image
> docker rmi [image]

- Remove images of specific pattern
> docker images -a | grep "[pattern]" | awk '{print $3}' | xargs docker rmi

- List containers
> docker container ls
OR
> docker ps

- List stopped containers
> docker ps --filter "status=exited"

- Run container
> docker run [image]

- Run new container interactively
> docker run -it [image]

- Run existing container interactively
> docker exec -it [container] bash

- Stop docker container running in the background
> docker stop [container]

- Remove docker container running in the background
> docker rm [container]

- Remove all stopped containers
> docker container prune

- Build image
> docker build -t [tag_name] [directory]

- Create network
> docker network create [name]

- Inspect network
> docker inspect [name]

- Check logs
> docker logs [name]
